# geokrety-service

GeoKrety micro services container

## Quick Start

 * to get available commands:
````
make
```` 
 * to build and run `geokrety-service` container:
 
```` 
make up
```` 

 * to tail `geokrety-service` logs:
 
```` 
make tail
```` 

 * to stop and remove container:
````
make stop
````
